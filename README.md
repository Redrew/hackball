HackBall
==============
Built from [Soccer.JS](https://github.com/Mati365/Soccer.js)

## Installing
**Build client:**
```
sudo npm install
```
```
npm run-script fix-gulp
```
and
```
./node_modules/.bin/gulp es6
```
**Run server:**
```
npm run-script start
```
Start the client by opening client/index.html in browser
